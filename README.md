📌 Proyecto — Proyecto HTML con estilos

Proyecto de maquetación web estática desarrollado con HTML, SASS y Vite. El objetivo es construir una landing educativa moderna con múltiples secciones de contenido, aplicando diseño responsive mobile-first, organización modular de estilos y control preciso del layout mediante Flexbox.

✨ Características

- Estructura semántica con header, main, section y footer
- Layout responsive mobile-first
- Organización modular de estilos con SASS (SCSS)
- Uso de Flexbox para distribución de contenido
- Tarjetas de cursos con composición visual estructurada
- Sección de equipo con presentación de perfiles profesionales
- Sección de blog con artículos destacados
- Navegación adaptada a distintos tamaños de pantalla
- Efectos hover en imágenes y elementos interactivos

🧩 Secciones del diseño

- Header con navegación principal y acceso de usuario
- Hero con mensaje principal y composición visual
- Sección de formaciones con tarjetas de cursos
- Sección de equipo docente
- Sección de artículos del blog
- Footer con redes sociales e información legal

🛠 Tecnologías utilizadas

- HTML5
- SASS (SCSS)
- Vite
- Flexbox
- Responsive Design
- GitHub Pages

🎯 Objetivo de aprendizaje

Desarrollar una landing completa aplicando maquetación responsive, control de layout con Flexbox, reutilización de estilos mediante SASS y estructuración modular del proyecto, reforzando la organización del código y la construcción de interfaces reales.
